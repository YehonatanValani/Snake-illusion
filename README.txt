1. The Snake Illusion Generator.py script does not require external files. you can edit the desired parameters, the destination folder for saving the images and the names you want to give them.
2. The Research Program.py script requires the following images to be in the same directory: opening_image, pause_image and pause_image2. You also need to edit the name of the answer file that will be saved, 
   Its position, and the presentation time for the random images can also be changed.
3. The Answers Sorter.py script sorts the results alphabetically. Requires entering the address and name of the answer file you want to sort and the address and name of the sorted file.
